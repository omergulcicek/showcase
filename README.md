# Showcase

A collection of demos and small projects built with **Next.js**, **React**, and **Tailwind CSS**.

## Projects

### [Calendar](/calendar)

Subscribe to curated event calendars (battles, leaders, nature, spiritual & cultural dates) and view them in your device’s calendar via ICS subscription.

- **Live:** [omergulcicek-calendar.vercel.app](https://omergulcicek-calendar.vercel.app/)
- **Stack:** Next.js, React, Tailwind CSS, Radix UI, Sonner

### [Forms](/forms)

Interactive documentation and demo for the [@omergulcicek/forms](https://www.npmjs.com/package/@omergulcicek/forms) npm package — smart masking and validation for React Hook Form with TypeScript.

- **Live:** [omergulcicek-forms.vercel.app](https://omergulcicek-forms.vercel.app/)
- **Package:** [npm](https://www.npmjs.com/package/@omergulcicek/forms) · [GitHub](https://github.com/omergulcicek/forms)
- **Stack:** Next.js, React Hook Form, @omergulcicek/forms, Motion, Tailwind CSS

### [Password Input](/password-input)

Interactive documentation and demo for the [@omergulcicek/password-input](https://www.npmjs.com/package/@omergulcicek/password-input) npm package — a React hook that adds a show/hide toggle to password inputs.

- **Live:** [omergulcicek-password-input.vercel.app](https://omergulcicek-password-input.vercel.app/)
- **Package:** [npm](https://www.npmjs.com/package/@omergulcicek/password-input) · [GitHub](https://github.com/omergulcicek/password-input)
- **Stack:** Next.js, @omergulcicek/password-input, Radix UI, Tailwind CSS
