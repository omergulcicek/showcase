# Introduction

**Password Toggle** is a professional React hook for password visibility toggle with accessibility and customization support. It handles only behavior and accessibility—styling is entirely up to you.

## Features

- 🔒 **Secure**: Handles password visibility toggle securely.
- ♿ **Accessible**: Full ARIA support for screen readers.
- 🎨 **Customizable**: Use custom icons or text for the toggle button.
- 📦 **Lightweight**: Zero dependencies (except React).
- 🧩 **Easy Integration**: Works seamlessly with any UI library (shadcn/ui, MUI, etc.).

## Installation

```bash
npm install @virastack/password-toggle
# or
yarn add @virastack/password-toggle
# or
pnpm add @virastack/password-toggle
```
